### Descripcion
Creacion de una pequeña aplicacion de consumo donde se muestran elementos de un JSON con precio y nombre
donde se calculara el costo del consumo por cada articulo asi como si este desea añadir porcentaje de alguna propina, este calcula el subtotal, el total de la propina y el total final o global. 


Deploy Link:https://propina-calc.netlify.app/
